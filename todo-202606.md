
待补齐：


# P3 远期目标

## 1. 多节点副本和分布式 shard

说明：

```text
属于架构级扩展，需要先稳定单节点存储引擎、恢复语义和 compaction 策略。
```

## 2. 在线一致性快照

说明：

```text
当前 backup / restore 已有校验和 pending restore，但仍偏离线目录级流程。
如要支持生产在线备份，需要设计快照屏障、WAL 截点和 manifest 一致性协议。
```

## 3. 更完整 InfluxDB 1.x 兼容层

说明：

```text
当前定位是兼容子集，不是完整替代。
远期可以按真实客户端和查询 corpus 决定是否扩展更多 InfluxQL 语法、函数和 HTTP 行为。
```
