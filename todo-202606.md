# MiniInflux.Net10 未完成功能清单

本文档只记录截至 2026-06 仍未完成或仅完成第一版的事项，按优先级排序，供后续开发排期使用。

优先级说明：

```text
P0：直接影响数据正确性、稳定性、线上可控性
P1：影响性能、兼容性、长期运行体验
P2：高级能力、运维便利性、长期演进方向
```

```sql

```

## 3. Subquery

目标能力：

```sql

  SELECT max(value) FROM cpu GROUP BY time(10s)
) GROUP BY time(1m)
```
